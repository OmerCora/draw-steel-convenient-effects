This is a fork DFreds Convenient Effects fork for Draw Steel System. 
Draw Steel have a special Active Effect type called Ability Modifiers.
This fork add support to save those effects in the Convenient Effects tab while also adding these additional features:
- Stackable toggle for each effect (For exmaple to track Werewolf's Accursed Rage)
- Temporary toggle for effect to able applied as temporary effects instead of passive effects
- An button on hover for effect indicator (top right) to print the effect description to chat.
